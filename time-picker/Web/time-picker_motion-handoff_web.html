Helper text

Al hacer focus o click en el field se abre un dropdown integrado con una lista de horarios usando Option Items. El usuario elige una hora, el valor queda en el field y el dropdown se cierra. No hay botones Cancelar ni Aceptar. Companion App: Time Picker App.

State: smart default · 09:30 AM
Token Mapping
  • Estados del field — hover, focus ring, error, pressed · motion-curve-md · easing-decelerate · 300ms
  • Panel enter — opacity + translateY (flip si no hay espacio) · motion-curve-md · 300ms
  • Panel exit — opacity + translateY · motion-exit · 200ms
  • Option Item filas — hover, pressed, selected · motion-curve-sm · 150ms · handoff enlazado
