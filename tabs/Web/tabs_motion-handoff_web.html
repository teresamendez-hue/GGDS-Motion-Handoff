Preview de motion

Motion lab — no es el componente de la librería. Validá motion y tokens; la UI final está en Figma.

Hacé click en los tabs o deslizá la lista horizontalmente. Cambiá variante Line / Segmented. Al elegir un tab fuera del viewport, la lista hace scroll para revelarlo. Checkbox: tab disabled y prefers-reduced-motion.

Token Mapping
Indicador + panel: motion-curve-md → easing-decelerate → cubic-bezier(0.0, 0, 0.2, 1) → duration-300 (300ms)
Tab item: motion-curve-sm → easing-standard → cubic-bezier(0.4, 0, 0.2, 1) → duration-150 (150ms)

Motion Specification

Tab con indicador Line o Segmented se desplaza con motion-curve-md. El panel hace slide horizontal con el mismo token. Estados del item con motion-curve-sm. Scroll horizontal nativo en overflow con reveal del tab activo.

Implementación CSS

.tabs__indicator {
  transition-property: transform, width;
  transition-duration: var(--motion-duration-from-motion-curve-md);
  transition-timing-function: var(--motion-curve-md);
}

.tabs__panel-track {
  transition-property: transform;
  transition-duration: var(--motion-duration-from-motion-curve-md);
  transition-timing-function: var(--motion-curve-md);
}

.tabs__tab {
  transition-property: color, opacity;
  transition-duration: var(--motion-duration-from-motion-curve-sm);
  transition-timing-function: var(--motion-curve-sm);
}

.tabs__tab:disabled { transition: none; }

@media (prefers-reduced-motion: reduce) {
  .tabs__indicator,
  .tabs__panel-track,
  .tabs__tab { transition: none; }
}
