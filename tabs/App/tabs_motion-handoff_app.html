Cómo probarlo: tocá un tab para cambiar de panel. Usá los radios Line / Segmented para cambiar variante. No hay hover en App; cada tap muestra el haptic simulado en el readout. El checkbox disableAnimations desactiva las transiciones.

Referencia visual Web: abrir preview Web

Este preview es una representación visual del motion, no una simulación física real. El navegador no soporta springs de Flutter. Los valores que el dev implementa son los parámetros de spring del Token Mapping.
