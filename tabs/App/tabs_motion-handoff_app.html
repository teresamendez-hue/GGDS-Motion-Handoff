Motion Specification

Tabs en App usa motion de estados internos, sin enter/exit de viewport: Enabled, Pressed, Focused, Selected, Disabled — no existe estado hover en App.

Pressed y Focused usan motion-spring-sm, igual que Button. El cambio de Selected mueve el indicador de posición con motion-spring-md, mismo token en ida y vuelta.

Disabled es instantáneo. Con MediaQuery.disableAnimations se aplica el estado final sin animar, incluido el indicador.

Implementación Dart (Flutter)

final motion = context.motionTokens;
final stateSpring = motion.springSm;      // motion-spring-sm (mass 1.0, stiffness 400, damping 35)
final indicatorSpring = motion.springMd;  // motion-spring-md (mass 1.0, stiffness 300, damping 28)
final disableAnimations = MediaQuery.of(context).disableAnimations;

void onTabSelected(int index) {
  if (disableAnimations) {
    indicatorController.value = targetOffsetFor(index);
    return;
  }
  indicatorController.animateWith(
    SpringSimulation(indicatorSpring, indicatorController.value, targetOffsetFor(index), 0),
  );
}

// pressed/focus animan scale/opacity con stateSpring (motion-spring-sm)
// disabled: estado final instantáneo, sin animar
