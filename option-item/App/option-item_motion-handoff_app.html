Este preview representa el motion del componente Option Item: la pieza base que consumen componentes de mayor jerarquía de resultados o selección (Combobox, Select, etc.). Badge, Thumbnail y Checkbox son opcionales dentro del ítem; si aplica, cada uno tiene su propio motion en su handoff.

Este preview es una representacion visual del motion, no una simulacion fisica real de Flutter. Implementa con los parametros del Token Mapping.

Implementacion Dart (Flutter)

const _optionItemSpring = SpringDescription(
  mass: 1.0,
  stiffness: 400.0,
  damping: 35.0,
);

void animateOptionItemState({
  required AnimationController controller,
  required double target,
  required BuildContext context,
}) {
  if (MediaQuery.of(context).disableAnimations) {
    controller.value = target;
    return;
  }
  controller.animateWith(
    SpringSimulation(
      _optionItemSpring,
      controller.value,
      target,
      0.0,
    ),
  );
}
