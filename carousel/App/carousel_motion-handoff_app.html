Motion Specification

Carousel en modo peek desplaza el track con motion-spring-md. Navegación por swipe y dots. Los dots animan el indicador activo con motion-spring-sm. Variantes infinite y finite, sin autoplay, sin flechas.

Implementación Dart (Flutter)

final springSlide = context.motionTokens.springMd;  // motion-spring-md
final springDot = context.motionTokens.springSm;    // motion-spring-sm
final disableAnimations = MediaQuery.of(context).disableAnimations;

PageController(viewportFraction: 0.82); // peek

void animateToSlide(int index) {
  if (disableAnimations) {
    _pageController.jumpToPage(index);
    return;
  }
  _slideController.animateWith(
    SpringSimulation(springSlide, _slideController.value, index.toDouble(), 0),
  );
}

// infinite: jumpToPage al slide real tras animar al clone

Haptics

Token Mapping
haptic-selection-change → haptic-selection-click → HapticFeedback.selectionClick()
Solo en tap de dot. Swipe: sin haptic. 
void onDotTapped(int index) {
  if (index != _currentPage) {
    context.haptics.trigger(GgdsHapticSemantic.selectionChange);
    animateToSlide(index);
  }
}
