Arrastrá el carousel, usá flechas o dots. Modo peek con slides adyacentes visibles. Cambiá la variante infinite / finite. Las flechas no animan acá — ver handoff de Icon Button.

Token Mapping
Slide: motion-curve-md → easing-decelerate → cubic-bezier(0.0, 0, 0.2, 1) → duration-300 (300ms)
Dots: motion-curve-sm → easing-standard → cubic-bezier(0.4, 0, 0.2, 1) → duration-150 (150ms)
Infinite reset: transition none (0ms) · Flechas: ver Icon Button

Implementación CSS

/* Track — cambio de slide (peek) */
.carousel__track {
  transition-property: transform;
  transition-duration: var(--motion-duration-from-motion-curve-md);
  transition-timing-function: var(--motion-curve-md);
}

/* Infinite — reset instantáneo al clonar */
.carousel__track--no-transition {
  transition: none;
}

/* Dots — indicador activo */
.carousel__dot {
  transition-property: transform, opacity;
  transition-duration: var(--motion-duration-from-motion-curve-sm);
  transition-timing-function: var(--motion-curve-sm);
}

@media (prefers-reduced-motion: reduce) {
  .carousel__track,
  .carousel__dot {
    transition: none;
  }
}
